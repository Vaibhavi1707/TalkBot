# TalkBot:Voice Enabled Chatbot                                                                                                                     
  It requires microphone of your system to run along with python packages such as:                                             
        a. Pyttsx3: Python's text to speech converter wrapper                                                                       
        b. webbrowser                                                                                                                  
        c. wikipedia                                                                                                               
        d. Speech Recognition                                                                                                       
        e. Pygame                                                                                                                    
        f. PyAudio                                                                                                                 
  All the above can be installed pip.                                                                                          
  Commmand:  pip install <package_name>                                                                                     
  pip can be installed using sudo apt install python-pip (Only on linux machines)                                                
  Apart from the above packages we also need the jack server.                                                                     
  To execute the code execute the following commands on command prompt:                                                         
  $jack_control start //starts the jack server                                                                                    
  $python app.py                                                                                                         
  Enter url on your browser to execute:                                                                                              
  localhost:7000/
